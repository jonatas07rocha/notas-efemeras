Manifesto do Produto

O QUE É Memória RAM Externa

Um espaço de triagem para pensamentos voláteis. Projetado para captura instantânea. Notas antigas (>24h) tornam-se translúcidas para indicar urgência.

O QUE NÃO É Um "Segundo Cérebro"

Esta ferramenta não substitui Notion, Obsidian ou Evernote. Não serve para armazenamento de longo prazo.
